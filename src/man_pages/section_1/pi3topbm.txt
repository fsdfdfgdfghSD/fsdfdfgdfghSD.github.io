NAME
       pi3topbm - convert an Atari Degas .pi3 file into a portable bitmap

SYNOPSIS
       pi3topbm [pi3file]

DESCRIPTION
       Reads an Atari Degas .pi3 file as input.  Produces a portable bitmap as output.

SEE ALSO
       pbmtopi3(1), pbm(5), pi1toppm(1), ppmtopi1(1)
