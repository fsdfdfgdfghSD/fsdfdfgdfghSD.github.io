NAME
       spctoppm - convert an Atari compressed Spectrum file into a portable pixmap

SYNOPSIS
       spctoppm [spcfile]

DESCRIPTION
       Reads an Atari compressed Spectrum file as input.  Produces a portable pixmap as output.

SEE ALSO
       sputoppm(1), ppm(5)
