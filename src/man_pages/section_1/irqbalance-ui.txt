NAME
       irqbalance-ui - user interface for irqbalance

SYNOPSIS
       irqbalance-ui

DESCRIPTION
       irqbalance-ui  provides  an ncurses-based textual user interface to irqbalance, a daemon responsible for IRQ distribution on Linux systems.  It shows how IRQs are distributed over CPUs at given moment, and allows one
       to setup of the sleep interval and the IRQ and CPU banning at runtime.

HOMEPAGE
       https://github.com/Irqbalance/irqbalance
