NAME
       byobu-launcher - Byobu Launcher

DESCRIPTION
       byobu-launcher is a simple application that will launch screen(1) in a byobu(1) configuration, reconnecting to an existing detached session (if available).

SEE ALSO
       screen(1), byobu(1)
