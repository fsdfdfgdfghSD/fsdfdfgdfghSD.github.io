NAME
       pbmtozinc - convert a portable bitmap into a Zinc bitmap

SYNOPSIS
       pbmtozinc [pbmfile]

DESCRIPTION
       Reads a portable bitmap as input.  Produces a bitmap in the format used by the Zinc Interface Library (ZIL) Version 1.0 as output.

SEE ALSO
       pbm(5)
