NAME
       wifi-status - monitor the wireless interface

SYNOPSIS
       wifi-status [INTERFACE]

DESCRIPTION
       wifi-status is a program that monitors a wireless interface.

       This is often useful when trying to associate with an access point.

SEE ALSO
       iwconfig(8), ip(8)
