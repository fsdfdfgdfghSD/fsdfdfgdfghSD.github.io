NAME
       sirtopnm - convert a Solitaire file into a portable anymap

SYNOPSIS
       sirtopnm [sirfile]

DESCRIPTION
       Reads  a Solitaire Image Recorder file as input.  Produces a portable anymap as output.  The type of the output file depends on the input file - if it's an MGI TYPE 17 file, a pgm file is written. If it's an MGI TYPE
       11 file, a ppm file is written.  The program tells you which type it is writing.
